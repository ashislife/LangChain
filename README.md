# LangChain Models

This repository demonstrates the core model components available in LangChain, including **LLMs**, **Chat Models**, **Embedding Models**, **Prompt Templates**, and **Structured Outputs**.

## Project Structure

```text
LangChain_Models/
├── LLMs/
├── chatModel/
├── embedded_Model/
├── prompts/
├── structured_output/
├── requirements.txt
├── test.py
└── .gitignore
```

## Features

- Large Language Models (LLMs)
- Chat Models
- Embedding Models
- Prompt Templates
- Structured Output
- LangChain Core Examples

## Prerequisites

- Python 3.10+
- LangChain
- langchain-core
- python-dotenv

## Installation

Clone the repository:

```bash
git clone https://github.com/ashislife/LangChain_Models.git
```

Navigate to the project directory:

```bash
cd LangChain_Models
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the test script:

```bash
python test.py
```

Explore the examples inside the respective directories.

## Directory Overview

| Directory | Description |
|----------|-------------|
| `LLMs/` | Examples demonstrating Large Language Models in LangChain. |
| `chatModel/` | Examples of Chat Model integration and usage. |
| `embedded_Model/` | Examples for generating and using embeddings. |
| `prompts/` | Prompt templates and prompt engineering examples. |
| `structured_output/` | Structured output using Pydantic, TypedDict, and JSON Schema. |
